【酷开网络简介】

 酷开网络，全称深圳市酷开网络科技有限公司，独立运营始于2014年，注册资本1亿元，法人代表王志国。酷开网络自主研发了智能电视操作系统，即“酷开系统”，是智能电视系统运营的技术平台。目前，酷开网络已经从互联网电视终端商转型为运营服务技术平台。

        酷开网络聚焦智能电视系统研发和智能电视运营增值服务的主营业务，涉及影视、广告、购物、游戏、教育、应用分发、音乐等业务。目前，酷开网络拥有智能电视激活终端数量3300多万，其中，自有终端2800万月活终端数量1700多万，日活终端数1200多万。

  2018年，酷开公司规划3年内智能电视激活终端规模总量突破1亿，保持行业第一的龙头地位。酷开网络先后接受来自爱奇艺、腾讯、百度的投资入股，目前市场估值95.909亿元，成为了OTT行业内第一家且唯一一家估值近百亿的独角兽公司。

 【酷开网络价值】

   

         酷开网络与深圳广播电影电视集团、安徽出版集团、东阳品格传媒、天威视讯共同出资成立影视公司。新影视公司未来将以打造原创影视精品为战略定位，充分发挥影视作品投资发行、版权营销、衍生品开发的完整产业链优势和效能，以提升影视内容生产能力为核心，依靠自主研发、开拓创新，增强影视作品的原创能力，打造国内一流的原创影视精品，在文化事业、文创产业上占据一席之地。

       百度宣布战略投资酷开公司总额10.55亿元人民币，百度持股比例为11%，成为酷开第二大股东，酷开市场估值达到95.909亿元。酷开系统将与百度DuerOS系统深度融合，探索客厅智慧之路，强化升级原有内容影音服务优势，扩展语音和搜索能力，扩展近一百项以上的功能，使电视机产品已经超越了原来的电视，成为智慧屏幕，成为通往整个互联网信息的重要大屏幕。此次投资之后，酷开成为了OTT行业内第一家且唯一一家估值近百亿的独角兽公司。这是酷开继引入爱奇艺、腾讯作为战略股东后，短短两年内再次引入BAT三巨头的百度作为战略股东，受到了媒体和市场的高度关注。

【酷开网络战略】

     2018年1月23日，酷开媒体见面会，酷开网络提出在未来的黄金三年里，着力打造亿级的终端俱乐部。将通过 N 计划，拓展更多终端；通过流量共享计划，拓展更多内容；通过客厅品牌 VIP 计划，拓展更多品牌。     

     2018年，酷开网络正式剥离电视业务，专注构建开放统一的超级智能系统生态，挖掘大屏商业价值。原酷开电视产品、供应链、业务、运营、市场、服务团队全部平移整合至创维全球电商中心，实行“创维+酷开”双品牌运作。

     未来，酷开网络通过吸纳优质合作伙伴，将电视打造成用户共享美好生活的载体，在满足用户需求的基础上，也为市场提供更多、更好的内容，进一步挖掘大屏商业价值。
